Notification Banner

User can change: "aria-label" to add-on the label of the role="status". If the content is empty, then the screen reader will read as "empty" as default. (It cannot be changed)

Case 1: No icon, has content, no aria-label

It will read as "Case 1 success message"

Case 2: No icon, has content, has customized aria-label "Number of items"

It will read as "Number of items, 2"

Case 3: Has icon, has content, no customized aria-label

It will read as "Case 3 success message with icon"

Case 4: Has icon, has content, has customized aria-label "Number of items"

It will read as "Number of items, 4"

Case 5: Has icon, no content, no customized aria-label

It will NOT read.

Case 6: Has icon, no content, has customized aria-label "Information"

It will read "Information"
